HalalVest is built strictly for educational purposes and historical data exploration only. It is NOT financial advice. Past performance does not predict future returns. Stock prices and Shariah compliance can change. Before any investment, consult a SEBI-registered advisor and verify halal status via Zoya / Islamicly / Tabadulat / halalstock.in.

Halal Verification

All 97 stocks are screened against AAOIFI Shariah principles — no banks, NBFCs, insurance, alcohol, tobacco, gambling, weapons, pork, or adult content. Verified via halalstock.in (NSECode-based screening) and cross-checked with Zoya for borderline cases.

Financial ratios checked: debt/assets < 33%, interest income < 5% of revenue. Re-screened quarterly.

Coverage-Weighted CAGR

Newer IPOs don't have 10 years of price history. We use coverage-weighted CAGR (Morningstar methodology): each stock's contribution to a portfolio's 5Y CAGR is scaled by how much of its 5 years of data we actually have. A 2-year-old stock counts as 40% of its weight in the 5Y bucket — honest math, no fake history.

Coverage % is shown in the modal so you know data quality at a glance.

Data Source

Prices: Yahoo Finance split-adjusted close, fetched annually (March 31 each year, 2001–2026) for CAGR returns, plus monthly adj-close series for risk statistics and chart trajectories. Returns and risk numbers computed live in-browser using JavaScript — no server backend, no API rate limits, no stale data.

Risk Statistics Methodology

Sharpe, Sortino, Std Dev, Max DD, Beta are computed live from monthly adjusted closes over each portfolio's common window — the overlap of months where every holding has data. Weights are re-normalised across covered tickers (same coverage-weighting philosophy as the CAGR). For a portfolio whose newest holding listed in 2023, the window starts in 2023.

Conventions: sample variance with n−1; monthly→annual scaling by ×12 for mean and ×√12 for std deviation; Rf = 6.5% (Indian 10Y G-Sec proxy); Sortino uses MAR = Rf/12 with downside deviation; Max Drawdown is walked on a real compounded monthly NAV; Beta is regressed against NIFTY 50 monthly returns over the same window.

The modal chart shows the actual monthly NAV path rebased to 100 — the dip you see in early-2020 portfolios is the real COVID drawdown, not a generated curve. The 6M return on the Overview tab is the real NAV ratio over the last six months. Earlier versions of HalalVest displayed hardcoded approximations for these five fields; they are now derived end-to-end from the same Yahoo data that powers the CAGRs.
