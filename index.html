About HalalVest
Shariah-compliant investing, demystified.

What is HalalVest?

HalalVest is a curated portfolio platform for Muslim investors in India who want to grow wealth without compromising their faith. Every stock in our 21 portfolios is screened against Shariah principles — no banks, NBFCs, alcohol, tobacco, gambling, weapons, or adult content.

HalalVest fetches fresh return data every session via Claude AI with web search. It queries Screener.in, Moneycontrol, and Tickertape for real CAGR figures and falls back gracefully to research-backed sector benchmarks when needed.

How the data engine works

On load, HalalVest calls the Claude AI API with web search enabled. Claude searches Indian financial databases and returns structured JSON with 1M, 6M, 1Y, 3Y, 5Y, and 10Y return data for key holdings. These are weighted by portfolio allocation and cached in your browser session — so it only runs once per tab.

Fallback: If the AI fetch fails or returns partial data, HalalVest uses research-backed sector benchmark ranges derived from 10+ years of NSE sector index data. These are clearly labelled "Research Est." in the UI.
